What is version control?

Version control is a system that tracks changes to files over time, allowing developers to revert to previous versions, collaborate efficiently, and manage different code versions.

Repo : Repo is a folder where git is initialized and where all the version control information is stored.

What is Git?

Git is a distr ibuted version control system that tracks changes in code, allowing multiple developers to collaborate efficiently.

What is GitHub?

Github is a cloud-based platform that hosts git repositories, enabling developers to collaborate, manage code, and track changes efficiently.
